Git-komentoja

    Tallenna muutokset GitHub repon master/main-haaraan:
  • git add -A
  • git commit -m"message goes here"
  • git push
    A: Lataa repository tietokoneelle kansioon:
  • git clone {https://github.com/...}
    B: Lataa ja yhdistä muutokset olemassa olevaan paikalliseen git-kansioosi:
  • git pull
