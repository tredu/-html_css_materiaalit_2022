  • Katso web-suunnittelijan luoma Figma-design: Figma
  • Vertaile elementtien asettelua työpöytä- ja mobiilikoossa

Tehtävä

  • Toteuta luonnoksen mukainen CV itsellesi HTML ja CSS tekniikoilla
  • Julkaise CV:si GitHub Pagesin avulla
  • Huomioi, että verkossa julkaistu CV on kaikkien käyttäjien nähtävillä:
    Älä kerro niitä (yhteys)tietoja joita et halua kaikkien tietävän.

Huomiot

  • Git on versionhallintajärjestelmä, joka mahdollistaa useiden ohjelmistokehittäjien työskentelyn saman projektin parissa.
  • Samalla Git pitää huolen siitä, ettei uudempaa koodia epähuomiossa korvata vanhalla.
  • Figma on käyttöliittymien suunnitteluun tarkoitettu ohjelma
  • Web-suunnittelija tekee usein sivustosta eri versiot työpöytä-, tabletti- ja mobiilikoossa
  • GitHub Pagesin avulla voi julkaista yksinkertaisia verkkosivuja
  • GitHub Pages soveltuu siten CV:n, portfolion tai muun demo-projektin julkaisuun
